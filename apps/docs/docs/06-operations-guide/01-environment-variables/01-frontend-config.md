# 웹 서버 환경 변수

## 필수 환경 변수

| 환경 변수                  | 설명                                  | 기본값 | 예시                    |
| -------------------------- | ------------------------------------- | ------ | ----------------------- |
| `NEXT_PUBLIC_API_BASE_URL` | 클라이언트 측에서 사용할 API 기본 URL | _필수_ | `http://localhost:4000` |

## 선택적 환경 변수

| 환경 변수              | 설명                    | 기본값 | 예시   |
| ---------------------- | ----------------------- | ------ | ------ |
| `NEXT_PUBLIC_MAX_DAYS` | 쿼리 최대 일수          | `90`   | `90`   |
| `PORT`                 | 웹 서버가 실행되는 포트 | `3000` | `3000` |
