# Overview

ABC User Feedback is a standalone web application designed to manage Voice of Customer (VoC) data. It enables you to efficiently gather and categorize customer feedback. The application is currently utilized in services with a reach of 10 million MAU.

## Technology Stack

ABC User Feedback uses the following technologies:

- **Frontend**: Next.js, React Query, React Hook Form, Tailwind CSS, MUI
- **Backend**: Node.js, NestJS, TypeORM
- **Database**: MySQL
- **Search Engine**: OpenSearch (optional)
- **Monorepo Management**: TurboRepo

- **Feedback Tag**: You can assign tags to each feedback to categorize them by topic.
- **Kanban Mode**: Experience the advantage of organizing and visualizing issue groups efficiently with Kanban mode.
- **Issue Tracker**: The Issue feature has a status indicator that lets you use it as a simple issue tracker. You can also link each issue to a ticket in your own issue tracker system.
- **Single Sign-on**: Authentication offers OAuth to accommodate enterprise-level single sign-on (SSO) requirements.
- **Role Management**: Role Based Access Control (RBAC) to effectively manage user permissions.
- **Dashboard**: Dashboard helps you visualize statistical data about your feedback and issues so you can learn about them at a glance.

## Deployment Options

ABC User Feedback can be deployed through the following methods:

1. **Docker Deployment** - Get started quickly using pre-built Docker images
2. **Build from Source** - Build directly from source code for customization
3. **Development Environment via CLI Tool** - Set up your development environment quickly using the `auf-cli` command-line tool
